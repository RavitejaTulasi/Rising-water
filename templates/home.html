Flood Prediction System

Flood forecasting is the use of rainfall and weather information to estimate the possibility of flooding. This Machine Learning based system analyzes rainfall, cloud cover, humidity and climatic conditions to predict whether flood conditions are likely to occur.

Predict Floods

About the Project

Floods are among the most destructive natural disasters. Early prediction helps governments and citizens prepare for emergencies.

This project uses Machine Learning algorithms trained on historical rainfall data to estimate flood occurrence using weather-related parameters.

Dataset

Historical rainfall records collected across Indian states.

Machine Learning

Random Forest Classifier trained on rainfall and weather features.

Prediction

Predicts whether flood conditions are likely based on user inputs.
